Exercise API

The Free Exercise API

After finding a lack of free easy-to-use resources for finding and creating exercises, I decided to create this API to hopefully make it easier for others.
This is by no means a finished project, but I am working on making it as functional as possible.

The application has a few parts:

The Exercise Database
A searchable interface to look up various exercise and get reference videos and more. Add in exercises that you can't find and even create a list of exercises to download.
The Workout Log
A minimalistic workout logging app to help you log your workouts undisturbed.
The API
More geared towards integrations with other applications. It will offer almost all of features above, just through an interface that can be used to access the data from other applications. Documentation to come.

Workout Log

Currenly in beta, I am also working on making a minimalist workout logger app that can assist people in logging their workouts. This logger is based off of the idea of being as simple as possible, but presented in a UI that's somewhat easier than an notebook or an Excel sheet.
